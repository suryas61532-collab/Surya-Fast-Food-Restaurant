# 🍔 Fast Food Restaurant Website

A responsive Fast Food Restaurant website built using **HTML5, CSS3, and JavaScript**. The project provides an attractive and user-friendly interface for customers to explore the restaurant menu, services, and contact information.

## 🚀 Features

- Responsive website design
- Attractive homepage with hero section
- Navigation menu
- Food menu display
- Services section
- About section
- Contact section
- Smooth scrolling
- Interactive buttons
- Modern UI design
